## JS를 이용한 Example

#### VanillaJS

> - Momentum

```
기능

1. 현재 시간을 보여준다
2. 이름을 입력할 수 있다.
3. 할 일에 대한 처리 (입력, 삭제) 할 수 있다.
```

### ReactJs

> - movie

```
기능

1. API를 이용해 영화 정보(이름, 포스터, 별점 etc)를 가져온다.
2. 특정 영화를 클릭할 경우 해당 영화 페이지(default)로 이동한다.
3. React Router를 이용한여 특정 영화 페이지로 넘어갈때 데이터를 넘겨준다.
```
